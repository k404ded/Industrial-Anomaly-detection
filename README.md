# Industrial Anomaly Detection using Machine Learning

## Overview
This project detects anomalous machine behavior using sensor data and Isolation Forest algorithm. It simulates predictive maintenance used in industrial environments.

## Dataset
AI4I 2020 Predictive Maintenance Dataset  
Contains:

- Air temperature
- Process temperature
- Rotational speed
- Torque
- Tool wear

## Technologies Used

Python  
Scikit-learn  
Seaborn  
Pandas  

## Workflow

1. Load industrial sensor dataset
2. Normalize sensor data
3. Train Isolation Forest model
4. Detect anomalies
5. Visualize anomaly patterns
